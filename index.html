Introduction to HTML & CSS

In this tutorial, you'll learn the basics of HTML and CSS. These are the building blocks of any website and are essential skills for web development.

HTML Basics

HTML stands for HyperText Markup Language. It is used to structure content on the web.

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, world!</h1>
</body>
</html>

In the example above, the <html> tag is the root of the document, <head> contains meta-information, and <body> contains the visible content.

CSS Basics

CSS stands for Cascading Style Sheets. It is used to style the HTML content, making it visually appealing.

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
}

The above CSS styles the body element, setting the font, background color, and text color.
